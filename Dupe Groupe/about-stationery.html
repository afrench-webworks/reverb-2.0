About the Sample Exploring ePublisher Stationery (DITA)
The Sample Exploring ePublisher Stationery was created by a Stationery designer using ePublisher Pro and a Stationery design project. The Stationery created by the Stationery Designer contains style and formatting information, settings, and serves as a set of reusable settings and rules that you can use to quickly and easily create a new ePublisher Express project and generate output.

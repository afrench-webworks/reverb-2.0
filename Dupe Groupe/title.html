Paragraph Styles (DITA)
Paragraph styles apply to entire paragraphs of text. Use paragraph styles if you want to apply a certain formatting to a particular section of your content.

DITA Styles Quick Reference
Body
All content that serves as the body of this document uses the Body paragraph style. For example, any text that is not part of a table, numbered or bulleted list should use the Body paragraph style.

Paragraph Styles (DITA)
Bullet, Bullet 2
The Bullet styles allow you to create unordered lists. The Bullet styles provide you with a level of indentation that matches exactly with your Body paragraph style. The Bullet2 style provides you with one level of indentation from the Body paragraph style.

DropDownEnd
The DropdownEnd marker should be inserted at the end of the text that you want to display in an expand/collapse section. This marker indicates the end of the content you want to display in an expand/collapse section.

Paragraph Styles (DITA)
GlossDef
The GlossDef style should be applied to the glossary definition for glossary terms. Apply the GlossDef style to any text that should appear in a popup window.
Parent topic 
Paragraph Styles (DITA)
GlossTerm
The GlossTerm style should be applied to any glossary term that will appear in a popup window. Content with the GlossTerm style should always precede the GlossDef style.

Paragraph Styles (DITA)
Heading1, Heading 2, and Heading 3
Heading styles can be used to label titles for major sections in source documents. Each heading style differs in font size and font weight.
Parent topic 
Paragraph Styles (DITA)
Indent1, Indent2
The Indent styles should be used when you want to apply a greater level of indentation to your Body style. Use Indent1 if you want to indent your content one level in from the Body style. Use Indent2 if you want to indent your content two levels in from the Body style.
Parent topic 
Paragraph Styles (DITA)
Number, Number 1
The Number styles allow you to create ordered lists. The first item in your list should use the Number1 style, and the content following the first item should use the Number style.
Parent topic 
Paragraph Styles (DITA)
RelatedLinks
The RelatedLinks style creates related topics links in generated output. Apply this style to a list of cross-references that you want to display as links in your generated output.

Paragraph Styles (DITA)
Title
The Chapter Title paragraph style allows you to identify new chapters in your document. This style uses autonumbering to append chapter numbers to your headings.

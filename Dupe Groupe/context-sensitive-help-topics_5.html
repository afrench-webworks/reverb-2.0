Context-Sensitive Help Topics (DITA) : Creating Your First Context-Sensitive Help Topic (DITA) : To create your first context-sensitive help topic
 
To create your first context-sensitive help topic
1. In the DITA topic, “context-sensitive-help-topics.xml” add a <prolog>/<metadata>/<othermeta> element.
2. Set the name attribute of the <othermeta> element to TopicAlias.
3. Set the content attribute of the <othermeta> element to IDH_CreatingYourFirstContextSensitiveHelpTopicDITA .
4. Save the DITA topic.
5. Generate output.
6. Double-click the Topics Report icon in the Output Explorer and verify that the id appears.

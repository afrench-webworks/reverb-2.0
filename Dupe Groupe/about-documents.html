About the Sample Exploring ePublisher Source Documents (DITA)
The Sample DITA Exploring ePublisher Source Documents contain paragraph and character styles, and marker types that you can use to explore ePublisher capabilities. For example, the sample DITA source documents and configuration contain sample paragraph styles, such as Heading 1 , Heading 2 , Title , and Body paragraph styles, and sample character styles, such as Bold and Command , that are common in many DITA configurations.
You can use these Sample DITA Exploring ePublisher Source Documents when you request an ePublisher evaluation in the following ways:
1. Follow the instructions in the Exploring ePublisher section in the ePublisher Evaluation Guide, ePublisher Writer Guide , or ePublisher Help and use the Sample Exploring ePublisher Source Documents and Stationery to create a sample project in ePublisher Express . After creating the sample project, you can generate output, see how the generated output displays, customize the output, and implement online features such as popup windows, expand/collapse sections, related topics links, and context-sensitive help topics.
2. You can replace the existing content in the sample DITA source documents with some of your own content, configure your own paragraph, character styles, and marker types, generate a result and review output.
For information on configuring your own styles using DITA source documents, please see the wiki topic, DITA Configuration , and the help topic, Preparing DITA Source Documents . For more information and step-by-step instructions on how to work with the Sample Exploring ePublisher Source Documents and Stationery , see the Exploring ePublisher section in the ePublisher Evaluation Guide , ePublisher Writer Guide , or ePublisher Help .

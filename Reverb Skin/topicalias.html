Marker Types (DITA)
Marker types allow you to embed information in your source documents and add functional elements to your generated output. For example, you can use markers to create popup windows and context-sensitive help topics.

ExpandCollapse
The ExpandCollapse style creates links that display or hide content in an expand/ collapse section in generated output. Apply this style to any text that you want to serve as a link for an expand/collapse section.

Marker Types (DITA)
Filename
The Filename marker should be inserted if you want to specify output filenames when ePublisher generated output. Specify the desired name for the file as the marker text for the marker.

Marker Types (DITA)
PopupEnd
The PopupEnd marker should be inserted at the text that will display in a popup window. This marker indicates the end of the text you want to display in a popup window.

Marker Types (DITA)
TopicAlias
The TopicAlias marker should be inserted in any topic that you want to use as a context-sensitive help topic. Specify the topic ID you wish to assign to the topic as the marker text for the marker.

To create your first related topics list
1. Create <related-links> section in your DITA source content.
2. Configure the ePublisher Stationery by customizing the default.wwconfig file using the version in this stationery as a guide..
3. Generate output.
4. Verify that this topic now displays a Related Topics button at the bottom of the topic in the generated output.
5. Verify that when you click on the related topics button, a list of Related Topics links displays that contains the links listed at the end of this section.

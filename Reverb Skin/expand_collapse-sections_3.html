Expand/Collapse Sections (DITA) : Creating Expand/Collapse Sections (DITA) : To create expand/collapse sections
 
To create expand/collapse sections
1. Create content that will serve as the link users click to expand or collapse as the content of the <title> element of a DITA <task> .
2. Make sure the Stationery configuration uses the container boundaries marked by the <steps> element to specify the end of the content to be expanded or collapsed.

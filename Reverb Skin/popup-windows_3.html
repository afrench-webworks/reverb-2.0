Popup Windows (DITA) : To create a popup window
 
To create a popup window
1. Create a xref element from the content that will serve as the popup link to an element in the content that you want to display in the popup window.
2. In the ePublisher Stationery, the GlossTerm and GlossDef paragraph styles have been configured to display as popup content.
